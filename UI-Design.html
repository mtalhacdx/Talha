Frequently Asked Questions

What's the difference between UI and UX design?

UX (User Experience) Design focuses on the overall feel of the product and how users interact with it (research, flows, usability). UI (User Interface) Design deals with the visual touchpoints users interact with (colors, typography, buttons). I specialize in both to deliver complete solutions.

What design tools do you use?

My core toolkit includes:

  • Figma (primary design tool)
  • Adobe Creative Suite (Photoshop, Illustrator)
  • Sketch
  • ProtoPie for advanced prototyping
  • UserTesting for research
  • Hotjar for analytics
Do you create design systems?

Yes, I develop comprehensive design systems including:

  • Style guides (colors, typography, icons)
  • Component libraries
  • Design tokens
  • Documentation for developers
  • Responsive layout grids
  • Accessibility guidelines
How do you ensure accessibility?

All my designs follow WCAG 2.1 guidelines:

  • Color contrast ratios (minimum 4.5:1)
  • Keyboard navigability
  • Screen reader compatibility
  • Clear visual hierarchy
  • Alternative text for images
  • Responsive design principles
What deliverables will I receive?

Typical deliverables include:

  • User research reports
  • User personas and journey maps
  • Wireframes and prototypes
  • High-fidelity mockups
  • Design system documentation
  • Developer handoff files
  • Style guides and assets
